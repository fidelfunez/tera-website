# Tera — Agencia de Desarrollo Web

Sitio web para Tera, agencia de desarrollo web en Honduras y Latinoamérica.

## Stack

- Next.js 14
- React 18
- Tailwind CSS
- Framer Motion
- TypeScript

## Desarrollo

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
